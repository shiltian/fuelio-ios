# Fuel - Car Fueling Tracker

A native iOS app built with SwiftUI and SwiftData to track your vehicle's fuel consumption, costs, and efficiency.

## Features

- **Multi-Vehicle Support** — Track multiple vehicles, each with its own dashboard and history
- **Statistics Dashboard** — View MPG, total spent, cost per mile, and trend charts
- **Smart Data Entry** — Auto-calculates the third field when you enter any two of: price, gallons, or total cost
- **CSV Import/Export** — Backup your data or migrate from other apps

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Getting Started

1. Open `Fuel.xcodeproj` in Xcode
2. Build and run on your device or simulator
